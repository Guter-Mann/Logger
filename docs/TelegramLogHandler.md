
# TelegramLogHandler

Класс наследуемый от `logging.Handler`, предназначен для регистрации
обработчика который пересылает логи в чат телеграма. Для отправки нужно
зарегистрировать телеграм бота в `@BotFather`. при создании нового бота нужно
обязательно активировать чат чтобы бот смог отправлять логи, для этого нужно
активировать скрипт и обработать команду `/start`

## Параметры

- **user_id**: `str | int` - Индентификатор пользователя (чата / канала куда
будет бот отправлять логи);
- **bot_token**: `str` - Токен телеграм бота. Его можно получить в `@BotFather`;
- **parse_mode**: `logger.enums.ParseMode` - Метод форматирования сообщения.
Данный формат так же подерживает все ключевые слова которые были предусмотренны
в `logger.formater` и поддерживает форматирования HTML, Markdown, MarkdownV2;
- **level**: `logging.LoggerLevel` - Уровень логов. Принимает числа от 10 до
50\. Можно использовать такой формат записи logging.INFO.
